# TAQ Data Access via WRDS

## Overview

This repository provides a structured workflow to access, clean, and analyze **TAQ (Trade and Quote) data** from **WRDS** using a modular approach inspired by the *tidyfinance* framework. The goal is to ensure reproducibility, scalability, and clarity in handling high-frequency market microstructure data. 
I build up upon their code to connect to WRDS, https://www.tidy-finance.org/

## Requirement
TAQ data subscription in WRDS
